EiGo プライバシーポリシー

AI英語学習アプリ EiGo のプライバシーポリシーです。

データの保存について

学習進捗、ユーザー設定等のデータは、お使いの端末内にのみ保存されます。アカウント登録は不要です。

サーバーとの通信について

本アプリでは、以下の機能をご利用の際にサーバーとの通信が発生します。

  • AI英会話・レッスン生成機能: AIによる英会話練習やレッスン生成時に、ユーザーの入力テキストが Google Gemini API に送信されます。送信されたデータはレッスン生成の目的でのみ使用され、サーバーに永続的に保存されることはありません。
  • 音声認識: 音声入力時に、OS標準の音声認識サービスとの通信が発生する場合があります。

上記以外の機能（プリセットレッスンの学習、進捗確認等）はオフラインで動作し、サーバーとの通信は発生しません。

収集する情報

本アプリは、個人情報を一切収集しません。メールアドレス、氏名、デバイスID、広告識別子等の情報は収集・送信されません。アナリティクスやトラッキングツールは使用していません。広告は表示されません。

第三者へのデータ提供

AI英会話機能ご利用時に、入力テキストが Google Gemini API で処理されます。Google の利用規約およびプライバシーポリシーが適用されます。それ以外の目的で、ユーザーデータを第三者と共有することはありません。

お子様のプライバシー

本アプリは全年齢を対象としていますが、13歳未満のお子様の個人情報を意図的に収集することはありません。
